GDP Analysis Engine
===================

A modular, plugin-based engine for loading, cleaning, and analysing GDP data
across countries and continents. The system exposes a programmatic core API,
a Streamlit dashboard, a REST output API, and a CLI — all wired together
through a shared plugin architecture.
